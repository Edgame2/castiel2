# Template Service Module

Template management service.

**Service**: `containers/template-service/`  
**Port**: 3037  
**API Base**: `/api/v1/templates`  
**Database**: Cosmos DB NoSQL (containers: `template_templates`, `template_context_templates`, `template_email_templates`)

## Overview

The Template Service module provides template management for context, email, and document templates.

## Features

- Template CRUD operations
- Context templates
- Email templates
- Document templates
