# Changelog

All notable changes to the Multi-Modal Service module will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added

- Initial implementation of Multi-Modal Service
- Image processing support (design, screenshot, diagram, wireframe, mockup)
- Diagram processing support (architecture, flowchart, UML, ER diagram, sequence, class diagram)
- Audio processing support (voice command, meeting, tutorial)
- Video processing support (tutorial, demo, screen recording)
- Whiteboard parsing support
- Text extraction (OCR)
- Code generation from media inputs
- Structured data extraction
- Analysis with detected elements, patterns, and suggestions
- Confidence scoring
- Support for URL and base64 data inputs
- Processing options (generateCode, extractText, transcribe, analyze)
- Metadata support (dimensions, duration, types)
- Async processing
- Job cancellation support
- Large file support (100MB body limit, 5-minute timeout)
- Tenant isolation
- JWT authentication
- OpenAPI documentation
