This directory contains the exhaustive list of all Castiel API endpoints.

## Files

| File | Description |
|------|-------------|
| [ENDPOINTS.md](./ENDPOINTS.md) | Full endpoint list with implementation status (Fully / Partial / Stub / Missing) |

## Status Legend

- **Fully** — Complete implementation; service logic, tests, UI integration
- **Partial** — Implemented but gaps (tests, config, X-Tenant-ID, docs)
- **Stub** — Placeholder; backend not wired or minimal logic
- **Missing** — Not implemented or not exposed via gateway
