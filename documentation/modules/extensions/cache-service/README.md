# Cache Service Module

Caching and cache management service.

**Service**: `containers/cache-service/`  
**Port**: 3035  
**API Base**: `/api/v1/cache`  
**Storage**: Redis

## Overview

The Cache Service module provides cache administration, optimization, and warming.

## Features

- Cache administration
- Cache optimization
- Cache warming
