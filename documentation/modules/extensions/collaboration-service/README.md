# Collaboration Service Module

Real-time collaboration features service.

**Service**: `containers/collaboration-service/`  
**Port**: 3031  
**API Base**: `/api/v1/collaboration`  
**Database**: Cosmos DB NoSQL (containers: `collaboration_conversations`, `collaboration_messages`)

## Overview

The Collaboration Service module provides real-time collaboration features and conversation management.

## Features

- Real-time collaboration
- Conversation management
